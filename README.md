# Bloomodoro

**Bloomodoro** is a gamified Pomodoro timer chrome extension that helps you stay focused by growing a garden. As you complete focus sessions, your garden thrives.

## Features

-   **Focus Garden**: Watch your plants grow in real-time as you focus. Different plants appear as you progress!
-   **Gamification**: Earn XP, level up, and unlock achievements.
-   **Night Mode**: Toggle between a sunny day and a magical night theme.
-   **Customizable Timer**: Set your preferred focus duration.
-   **Notifications**: Get notified when your focus session ends.

## Installation

1.  Clone this repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Build the project:
    ```bash
    npm run build
    ```
4.  Open Chrome and navigate to `chrome://extensions/`.
5.  Enable **Developer mode** (top right toggle).
6.  Click **Load unpacked** and select the `dist` folder generated in the project directory.
